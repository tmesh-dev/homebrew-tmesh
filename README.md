# Tmesh-dev Tmesh

## How do I install these formulae?

`brew install tmesh-dev/tmesh/<formula>`

Or `brew tap tmesh-dev/tmesh` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tmesh-dev/tmesh"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
